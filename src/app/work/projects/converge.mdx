## Overview

Converge translates conversation message formats between the major LLM providers — OpenAI, Anthropic, and Google Gemini — entirely locally. Switching providers, or running the same conversation across several models, no longer means hand-reformatting message structures.

## The Problem

Every provider uses a slightly different, incompatible message schema. Teams working across multiple LLM APIs waste time reconciling these formats, and the edge cases — tool calls, system messages, refusals, image and audio content — are exactly where ad-hoc converters break.

## Key Features

- **Bidirectional conversion** across three major formats
- **Fully local** — no API calls, no auth, no network
- **Inspectable canonical format** — converts through an intermediate representation you can debug or modify
- **Handles the hard cases** — tool use, system messages, refusals, image and audio content
- **Modern message support** — keeps pace with current provider features

## Technology

TypeScript with zero runtime dependencies. Runs on Node 18+, Deno, Bun, or any ES2020 environment. Published on npm as `@jeremysnr/converge`, MIT licensed.
