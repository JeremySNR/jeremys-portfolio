## Overview

The Amadeus Flight Tool is a serverless API wrapper that gives voice assistants built with Vapi the ability to search for flights through the Amadeus Flight Search API. Callers query flight options by airport and date in natural conversation, and get structured results back.

## Key Features

- **Voice-integrated search** — designed to slot directly into Vapi voice assistants
- **Flexible queries** — one-way and return flights, configurable passenger counts and currencies
- **Test and production modes** — switch between the Amadeus sandbox and live environments
- **Structured responses** — pricing, departure/arrival times, and layover detail
- **Serverless deployment** — quick to host on Vercel

## Technology

TypeScript on Node 18+, deployed to Vercel, integrating the Amadeus Flight Search API with environment-based credentials and sample Vapi tool configuration.
